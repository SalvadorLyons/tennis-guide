## 一、计分与规则 (Scoring & Rules)

| 术语 | 含义 | 备注 |
|------|------|------|
| **Love** | 0分 | 源自法语 l'oeuf (蛋)，形似0 |
| **15 / 30 / 40** | 1/2/3分 | 赢下第4分即赢下这一局 (Game) |
| **Deuce** | 40-40 (平分) |此时需要连赢两球才能保发/破发 |
| **Ad (Advantage)** | 占先 | Deuce后赢一球。Ad-In(发球方占先)，Ad-Out(接发方占先) |
| **Tie-break** | 抢七 | 局分6-6时进行的决胜局，谁先到7分(且净胜2分)谁赢 |
| **Set / Match** | 盘 / 场 | 通常赢得6局为一盘，赢得2盘或3盘为一场 |
| **Let** | 重发 | 发球擦网但落入界内，需要重新发球 |
| **Fault / Double Fault** | 失误 / 双误 | 发球出界或下网。两次失误直接丢分 |

## 二、技术动作 (Technique)

| 术语 | 含义 | 备注 |
|------|------|------|
| **Topspin** | 上旋 | 现代网球核心。球落地后高弹、前冲 |
| **Slice** | 切削 / 侧旋 | 拍面从上往下切球。球落地不弹、向后滑 |
| **Flat** | 平击 | 几乎没有旋转的击球，速度快但容错率低 |
| **Volley** | 截击 | 球未落地直接在空中击回 |
| **Smash** | 高压 | 像发球一样把空中的高球扣杀回去 |
| **Drop Shot** | 放小球 | 假装抽球，实际卸力让球刚过网就落地 |
| **Lob** | 挑高球 | 把球挑过网前人的头顶，落到底线 |

## 三、战术与状态 (Tactics & Status)

| 术语 | 含义 | 备注 |
|------|------|------|
| **Winner** | 制胜分 | 凭借自己的好球直接得分，对手没碰到球 |
| **UE (Unforced Error)** | 非受迫性失误 | 没人逼你，自己打丢了 (菜) |
| **Ace** | Ace球 | 发球直接得分，对手拍子没碰到球 |
| **Cross Court** | 斜线 | 从左区打到右区，或反之 |
| **Down the Line** | 直线 | 沿着边线打 |
| **Hold** | 保发 | 赢下自己发球的那一局 |
| **Break** | 破发 | 赢下对手发球的那一局 |
| **Moonball** | 月亮球 | 又高又慢的球，虽然难看但对付急躁的新手很有效 |
| **Shank** | 呲了 | 球打在拍框上，乱飞 |
